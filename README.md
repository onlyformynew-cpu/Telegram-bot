Telegram бот для моніторингу графіка електропостачання
Цей бот дозволяє:

Додавати адреси для моніторингу
Перевіряти статус електропостачання
Автоматично надсилати повідомлення про зміни кожні 10 хвилин
Команди
/addaddress <місто> <вулиця> <будинок>` — додає адресу для моніторингу
/listaddresses — показує список збережених адрес
/deleteaddress <номер> — видаляє адресу за номером зі списку
/status <номер> — перевіряє статус електропостачання для конкретної адреси
/status all — перевіряє всі збережені адреси
Автоматичне відстеження
Бот автоматично перевіряє всі адреси кожні 10 хвилин. Якщо виявлено зміни — надсилає повідомлення та скріншот графіка.
